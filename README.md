# Unit Converter

[English](README-en.md) | [中文]

一个前后端分离的单位转换器，使用 Go、Gin、Vue 创建，支持多种单位转换

这是一个来自 [roadmap.sh](https://roadmap.sh/projects/unit-converter) 的后端练习项目。

## 快速开始

- 使用 `git clone` 克隆项目到本地

对于前端

- 进入 `\frontend` 目录，运行 `npm install` 安装依赖
- 执行 `npm run dev` 启动前端开发服务器
- 或使用 `npm run build` 构建生产版本

对于后端

- 进入 `\backend` 目录，运行 `go mod tidy` 安装依赖
- 执行 `make run` 启动后端服务器
- 或使用 `make build` 构建生产版本 （Windows 用户查看 `Makefile` 中的命令）

## 练习内容

- Gin 框架的基本使用
- RESTful API 设计
- 与第三方 API（如货币汇率）的集成
- 前后端分离的开发模式
